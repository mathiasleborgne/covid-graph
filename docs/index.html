This website shows the number of cases and deaths of Covid-19 by country (last update: {{ site.data.global_info.date_last_update }}), tries to make predictions with an algorithm.

As growth is currently exponential (see explanation here), they are displayed by default in a logarithmic manner, which means that each additional unit on the y-axis makes the number 10 times bigger. Curves are compared to:

  • an exponential curve (when it doesn't look flat yet)
  • or a logistic curve (when it's already starting to flatten).
  • or a logistic, then exponential curve (when the peak seems past).
The model with smallest error is chosen. Prediction is thus made, with an error rate (hoping that the numbers are overrated...). Only countries with more than {{ site.data.global_info.min_cases }} cases are shown. Please note that the predictions are made by an algorithm, no manual corrections are performed, and the algorithm can make mistakes for some cases.

Data source: ECDC - Code: github.com/mathiasleborgne/covid-graph

This website shows the number of cases and deaths of Covid-19 by country (last update: {{ site.data.global_info.date_last_update }}).

As growth is currently exponential (see explanation here), they are displayed in a logarithmic manner, which means that each additional unit on the y-axis makes the number 10 times bigger. Curves are compared to an exponential curve, with an error rate... which will hopefully increase in the near future as the curves flatten. Only countries with more than {{ site.data.global_info.min_cases }} cases are shown.

Data source: ECDC - Code: github.com/mathiasleborgne/covid-graph
